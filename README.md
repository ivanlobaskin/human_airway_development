# human_airway_development
Analysis and simulation code for "Developmental dynamics of the human bronchial tree" by Inoue et al.
lungs.nb is a Wolfram Mathematica notebook containing code for network analysis and network growth simulations. Note: Mathematica 14.3 is needed for full functionality.
stochastic_full_tree.py contains Python code to run model of branching dynamics. Simulation output can be exported as a csv file for analysis in lungs.nb
